# DONATE PROJECT

## Requirements
- PHP 7.x,
- Composer

## Instalation

-   Clone project
```
    git clone https://github.com/buihoang91tin/donate/
```
-   Cd in project
```
    cd donate
``` 
-   Install library
```
    composer install
```
-   Setup database
```
    php arisan migrate
```
-   Create db seed
```
    php artisan db:seed
```
-   Setup Passport API
```
    php artisan passport:install
```
-   Run server
```
    php artisan serve
```
